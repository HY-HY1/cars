# THE CAR FLIPPING PLAYBOOK — COURSE SPEC

## OBJECTIVE

Build a premium beginner-focused UK car flipping course delivered through the existing lesson component system.

The course must feel:
- Practical
- Tactical
- Real-world
- Example-heavy
- Beginner-friendly
- Professional but not corporate
- Clear and concise

This is NOT an academic course.
This is NOT motivational fluff.
This is NOT generic entrepreneurship content.

The course should feel like:
"An experienced flipper showing a beginner exactly how to avoid mistakes and make profit."

---

# CONTENT STYLE RULES

## Tone
- Clear
- Direct
- Practical
- No hype
- No fake guru language
- No exaggerated claims
- No "get rich quick" framing

## Writing style
- Short paragraphs
- Actionable advice
- Real examples
- UK terminology
- Explain WHY, not just WHAT
- Avoid filler

## DO NOT:
- Write giant walls of text
- Repeat ideas
- Use motivational fluff
- Overcomplicate explanations
- Pretend the business is risk-free

---

# LESSON STRUCTURE RULES

Every lesson should contain MOST of the following:

1. LessonHeader
2. LessonVideo
3. Multiple LessonSection blocks
4. LessonCallout blocks
5. LessonChecklist or LessonSteps where useful
6. LessonKeyTakeaway
7. LessonResourceGroup
8. LessonNavigation

Lessons should feel visually structured and broken into digestible chunks.
